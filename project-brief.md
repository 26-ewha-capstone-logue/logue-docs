# Logue

## Project Description
Logue는 비데이터 직군 사용자가 자연어로 질문하면,  
사전에 검증된 지표 체계를 기반으로 신뢰 가능한 분석 결과를 제공하는 웹 기반 데이터 분석 서비스입니다.

## Team
- team19

## Tech Stack
- Frontend: React
- Backend: Java / Spring Boot
- AI: LLM 기반 질의 해석 및 Metric 매핑
- Database: DuckDB
- Visualization: Recharts
