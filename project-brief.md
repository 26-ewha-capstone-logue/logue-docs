# Logue

## Project Description
Logue는 비데이터 직군 사용자가 자연어로 질문하면,  
사전에 검증된 지표 체계를 기반으로 신뢰 가능한 분석 결과를 제공하는 웹 기반 데이터 분석 서비스입니다.

## Team
- Team 19

## Tech Stack
- Frontend: React
- Backend: Java / Spring Boot
- AI: LLM 기반 질의 해석 및 Metric 매핑
- Database: DuckDB
- Visualization: Recharts

## Target User
- 데이터 분석 비전문 실무자
- 데이터팀 의존 없이 빠르게 지표를 확인하고 싶은 조직 구성원

## Core Features
- 자연어 기반 데이터 질의
- Metric Registry 기반 검증된 분석 실행
- 테이블 및 차트 결과 제공
- 지표 정의 및 계산 방식 설명
- CSV 업로드 기반 분석
